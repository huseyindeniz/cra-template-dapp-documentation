# Tools

## Start app in dev mode

Vite

```bash
npm run dev
```

CRA

```bash
npm run start
```

## Check linting problems

```bash
npm run lint
```

## Run tests

```bash
npm run test
```

## Check test coverage

```bash
npm run coverage
```

## Start Storybook

```bash
npm run storybook
```

## Extract strings to be translated

```bash
npm run extract
```
